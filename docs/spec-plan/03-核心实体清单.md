## 1. 基础实体

### 1.1 BaseEntity

**用途**: 所有实体的基类，提供通用字段和软删除功能

**所属包**: @oksai/core

**文件位置**: `libs/core/src/lib/entities/base.entity.ts`

**字段**:

| 字段名          | 类型    | 必填 | 默认值            | 描述       |
| --------------- | ------- | ---- | ----------------- | ---------- |
| id              | UUID    | 是   | gen_random_uuid() | 主键       |
| isActive        | Boolean | 否   | true              | 是否激活   |
| isArchived      | Boolean | 否   | false             | 是否归档   |
| archivedAt      | Date    | 否   | null              | 归档时间   |
| createdAt       | Date    | 否   | getCurrentDate()  | 创建时间   |
| updatedAt       | Date    | 否   | getCurrentDate()  | 更新时间   |
| deletedAt       | Date    | 否   | null              | 软删除时间 |
| createdByUserId | UUID    | 否   | null              | 创建人 ID  |
| updatedByUserId | UUID    | 否   | null              | 更新人 ID  |
| deletedByUserId | UUID    | 否   | null              | 删除人 ID  |

**索引**:

-   id (PRIMARY KEY)
-   isActive
-   isArchived

**特殊特性**:

-   软删除: 使用 `mikro-orm-soft-delete`
-   时间戳: 自动管理 createdAt 和 updatedAt

## 2. 系统实体

### 2.1 Tenant (租户)

**用途**: 租户，多租户核心实体

**所属包**: @oksai/tenant

**文件位置**: `libs/tenant/src/lib/entities/tenant.entity.ts`

**继承**: BaseEntity

**字段**:

| 字段名                          | 类型    | 必填 | 默认值            | 描述                |
| ------------------------------- | ------- | ---- | ----------------- | ------------------- |
| name                            | String  | 是   | -                 | 租户名称 (唯一)     |
| logo                            | String  | 否   | null              | 租户 Logo URL       |
| standardWorkHoursPerDay         | Number  | 否   | 8                 | 标准工作时长 (1-24) |
| imageId                         | UUID    | 否   | null              | 图片 ID (外键)      |
| isActive                        | Boolean | 否   | true              | 是否激活            |
| createdAt, updatedAt, deletedAt | -       | -    | 继承自 BaseEntity |

**关系**:

-   image: ManyToOne → ImageAsset
-   organizations: OneToMany → Organization
-   rolePermissions: OneToMany → RolePermission
-   featureOrganizations: OneToMany → FeatureOrganization

**索引**:

-   name (UNIQUE)
-   isActive

**业务规则**:

-   name 必须全局唯一
-   standardWorkHoursPerDay 范围: 1-24
-   租户被删除后，其下所有组织、用户、数据应被级联删除

### 2.2 User (用户)

**用途**: 用户，认证核心实体

**所属包**: @oksai/user

**文件位置**: `libs/user/src/lib/entities/user.entity.ts`

**继承**: TenantBaseEntity

**字段**:

| 字段名    | 类型    | 必填 | 默认值 | 描述            |
| --------- | ------- | ---- | ------ | --------------- |
| email     | String  | 是   | -      | 邮箱 (全局唯一) |
| password  | String  | 否   | null   | 加密密码        |
| firstName | String  | 否   | null   | 名              |
| lastName  | String  | 否   | null   | 姓              |
| imageUrl  | String  | 否   | null   | 头像 URL        |
| isActive  | Boolean | 否   | true   | 是否激活        |
| verified  | Boolean | 否   | false  | 邮箱是否验证    |
| roleId    | UUID    | 否   | null   | 角色 ID (外键)  |
| tenantId  | UUID    | 否   | null   | 租户 ID (外键)  |

**关系**:

-   role: ManyToOne → Role
-   tenant: ManyToOne → Tenant
-   invites: OneToMany → Invite
-   organizations: ManyToMany → Organization

**索引**:

-   email (UNIQUE)
-   isActive
-   tenantId

**业务规则**:

-   email 必须全局唯一

### 2.6 Invite (邀请)

**用途**: 用户邀请

**所属包**: @oksai/user

**文件位置**: `libs/user/src/lib/entities/invite.entity.ts`

**继承**: TenantOrganizationBaseEntity

**字段**:

| 字段名          | 类型   | 必填 | 默认值  | 描述             |
| --------------- | ------ | ---- | ------- | ---------------- |
| email           | String | 是   | -       | 邀请邮箱         |
| fullName        | String | 否   | null    | 邀请全名         |
| token           | String | 是   | -       | 邀请令牌 (唯一)  |
| status          | Enum   | 是   | INVITED | 邀请状态         |
| expireDate      | Date   | 是   | -       | 过期时间         |
| actionDate      | Date   | 否   | null    | 操作时间         |
| code            | String | 否   | null    | 验证码           |
| roleId          | UUID   | 否   | null    | 角色 ID (外键)   |
| userId          | UUID   | 否   | null    | 用户 ID (外键)   |
| invitedByUserId | UUID   | 是   | -       | 邀请人 ID (外键) |

**枚举**: InviteStatus

-   INVITED: 已邀请
-   ACCEPTED: 已接受
-   REJECTED: 已拒绝
-   EXPIRED: 已过期

**关系**:

-   role: ManyToOne → Role
-   user: ManyToOne → User
-   invitedByUser: ManyToOne → User
-   tenant: ManyToOne → Tenant
-   organization: ManyToOne → Organization
-   projects: ManyToMany → OrganizationProject
-   teams: ManyToMany → OrganizationTeam
-   departments: ManyToMany → OrganizationDepartment
-   organizationContacts: ManyToMany → OrganizationContact

**索引**:

-   email (复合索引: tenantId + email)
-   token (UNIQUE)
-   status
-   tenantId
-   expireDate

**业务规则**:

-   token 必须全局唯一
-   邀请链接有效期: 7 天 (可配置)
-   邀请被接受后，创建用户和组织关联
-   过期的邀请不能被接受

**用途**: 记录系统操作，用于审计和安全追踪

**所属包**: @oksai/audit

**文件位置**: `libs/audit/src/lib/entities/audit-log.entity.ts`

**继承**: TenantBaseEntity

**字段**:

| 字段名         | 类型   | 必填 | 默认值 | 描述                              |
| -------------- | ------ | ---- | ------ | --------------------------------- |
| action         | String | 是   | -      | 操作类型 (CREATE, UPDATE, DELETE) |
| entity         | String | 是   | -      | 实体名称                          |
| entityId       | UUID   | 否   | null   | 实体 ID                           |
| userId         | UUID   | 是   | -      | 操作人 ID (外键)                  |
| organizationId | UUID   | 否   | null   | 组织 ID (外键)                    |
| oldValue       | JSON   | 否   | null   | 旧值 (序列化)                     |
| newValue       | JSON   | 否   | null   | 新值 (序列化)                     |
| ipAddress      | String | 否   | null   | IP 地址                           |
| userAgent      | String | 否   | null   | 用户代理                          |

-   entity

-   tenantId

-   记录所有关键操作 (创建、更新、删除用户、组织、角色等)

-   审计日志不可修改或删除
